# staged

staged is a virtual staging app for real estate listings. users sign in with email, upload room photos, choose a style, and generate staged images for listings. the product is built to keep the workflow simple and fast for agents who need polished marketing photos.

the app uses cursor cloud agents for image generation, stripe for paid credit packs, and sqlite for local data storage. each account starts with free watermarked previews, and paid credits unlock clean full-resolution downloads.

everything is designed to run on a small low-cost setup while still supporting auth, image processing, rendering jobs, and checkout flow in one project.
